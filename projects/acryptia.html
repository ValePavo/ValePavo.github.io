My Technical Contributions

Inventory System (Web3 Integrated)

Created a flexible inventory system that interfaces directly with blockchain smart contracts. This solution allows for secure item ownership and trading empowering players to fully possess, exchange, and enhance their assets within the Acryptia ecosystem

Server-Authoritative Networking

Developed a server-driven logic layer using PlayFab and Mirror. All essential operations—such as deck creation, shuffling, and match validation—are handled exclusively on the server. Every player move is validated server-side to guarantee security, prevent exploits, and ensure fair gameplay for all participants

Scalable Card Logic

Utilized the Strategy Design Pattern to separate card behaviors from the main execution flow. This approach improved code maintainability and made it easy to introduce new card mechanics without modifying core systems

Dynamic Skinning System

Developed a custom RGB-mask shader solution for UI customization, enabling real-time recoloring of UI elements through ScriptableObjects. This approach significantly reduced build size by removing the need for duplicate textures for each skin variant
